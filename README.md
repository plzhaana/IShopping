# IShopping

## 项目介绍

  一款移动端商城购物App，商品数据采用网上的某博主的接口，据说服务器部署在腾讯云。项目主要分为主页、分类、购物车和“我的”四大模块。

## 特点

- 采用前端 vue 框架进行开发
- 采用 MVVM 思想架构
- 组件化开发，减少模块间的耦合
- 采用 promise + axios 进行网络请求
- 使用 vue-router 进行页面路由和 vuex 进行状态管理
